Line Generator¶

Things and plots

class scanpointgenerator.LineGenerator(name, units, start, step, num)[source]¶

Generate equally spaced scan points in one dimension

Args: name (str): The scannable name. E.g. “x” units (str): The scannable units. E.g. “mm” start (float): The first position to be generated. E.g. 1.5 step (float): The increment for each successive position. E.g. 0.5 num (int): The number of points to generate. E.g. 5

Examples¶

from scanpointgenerator import LineGenerator, plot_generator

gen = LineGenerator("x", "mm", 0, 0.1, 5)
plot_generator(gen)
